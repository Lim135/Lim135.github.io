¶easy_RSA

题目：
在一次RSA密钥对生成中，假设p=473398607161，q=4511491，e=17
求解出d

题目描述：没用，就硬做

¶RSA介绍

前置知识储备：质数、互质、模反元素、欧拉函数、欧拉定理、费马小定理（欧拉特殊情况），具体后续在进行介绍

步骤 说明 描述 备注
1 找出质数 P、Q -
2 计算公共模数 N=P * Q -
3 欧拉函数 φ(N) = (P-1)(Q-1) -
4 计算公钥E 1 < E < φ(N) E的取值必须是整数
E 和 φ(N) 必须是互质数
5 计算私钥D E * D % φ(N) = 1 -
6 加密 C = ME mod N C：密文 M：明文
7 解密 M = CD mod N C：密文 M：明文

公钥＝(E , N)
私钥＝(D, N)
对外，我们只暴露公钥。

¶解题步骤

由于本题数值太大，不方便使用上述方法手算，所以考虑使用python代码

编写python：

import gmpy2 

p = 473398607161 
q = 4511491 
e = 17 
phi = (p - 1) * (q - 1) 
d = gmpy2.invert(e, phi)  ##求逆元，d^e = 1 mod phi

print(d)

得出结果：125631357777427553

¶easychallenge

题目： 一个.pyc文件

题目描述： 没有意义的一串文字

因为没有任何提示，所以走一步看一步

¶解题步骤

  1. .pyc为.py的编译过后的文件，所以我们需要对.pyc文件进行反编译，得到原文
import base64

def encode1(ans):
    s = ''
    for i in ans:
        x = ord(i) ^ 36
        x = x + 25
        s += chr(x)
    
    return s


def encode2(ans):
    s = ''
    for i in ans:
        x = ord(i) + 36
        x = x ^ 36
        s += chr(x)
    
    return s


def encode3(ans):
    return base64.b32encode(ans)

flag = ' '
print 'Please Input your flag:'
flag = raw_input()
final = 'UC7KOWVXWVNKNIC2XCXKHKK2W5NLBKNOUOSK3LNNVWW3E==='
if encode3(encode2(encode1(flag))) == final:
    print 'correct'
else:
    print 'wrong'

由题目不难看出，这个文件给出了final，需要通过final逆推出本题的flag

  1. 修改代码顺序，加密过程上下顺序颠倒，运算逆置，加密解密顺序逆置，得到代码如下：
import base64

def encode1(ans):
    s = ''
    for i in ans:
        x = ord(i) - 25
        x = x ^ 36  # ^为异或运算，异或运算的逆运算还是异或运算
        s += chr(x)
    
    return s


def encode2(ans):
    s = ''
    for i in ans:
        x = i ^ 36 #ord()这个函数接受的类型是一个长度为1的字符串，而在python3当中传入的i已经是一个整型数了，直接用i^36就可以完成操作
        x = x - 36
        s += chr(x)
    
    return s


def encode3(ans):
    return base64.b32decode (ans)  #用 Base32 编码字节类对象（bytes-like object）ans，并返回编码过的字节（bytes），encode为加密，decode为解密

flag = ' '
final = 'UC7KOWVXWVNKNIC2XCXKHKK2W5NLBKNOUOSK3LNNVWW3E==='
flag = encode1(encode2(encode3(final)))
print(flag)

得出结果：cyberpeace{interestinghhhhh}
